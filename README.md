# Project C - AI Consulting Slide Generator

Project C is an intelligent full-stack application designed to automate the creation of high-quality consulting slides. It leverages strict architectural patterns ("Architect" -> "Designer" -> "Corrector" workflow) powered by the Anthropic Claude API to generate structured, visually appealing HTML/CSS slides from simple natural language prompts.

    ```properties
    # Replace with your actual key or set as environment variable
    anthropic.api.key=sk-ant-api03-...
    ```

    - _Note: For security, it is recommended to set `ANTHROPIC_API_KEY` as a system environment variable instead of hardcoding it._

3.  Ensure MongoDB is running. The app expects it at `mongodb://localhost:27017/projectc` by default.

4.  Run the application in Dev Mode (Hot Reload):
    ```bash
    ./mvnw quarkus:dev
    ```
    The backend will start on **http://localhost:8080**.

### 2. Frontend Setup (Client)


2.  Install dependencies:

3.  Start the development server:
    ```bash

    ```
    The frontend will launch typically on **http://localhost:5173**.
